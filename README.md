# Sistema_ERP_Sweet_Box

O projeto consiste na criação de uma aplicação de gestão para pequenos negócios. São desenvolvidas desenvolver boas práticas de desenvolvimento de software aliadas à Segurança da Informação, utilizando obrigatoriamente a linguagem Java.
Projeto do terceiro semestre de Análise e Desenvolvimento de Sistemas da Universidade de Mogi das Cruzes.
